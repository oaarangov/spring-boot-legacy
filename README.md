# Spring Boot Legacy Proyect
Spring Boot project with legacy dependencies version.

## Dependencies

* Spring Boot 2.7.18
* Gradle 7.2
* Java 11

## How to run
On the root folder run:
* Linux:
    * ./gradlew clean build
    * ./gradlew bootRun
* Windows:
    * gradlew.bat clean build
    * gradlew.bat bootRun
